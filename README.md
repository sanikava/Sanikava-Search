# Sanikava-Search
Search Engline made in nodejs TypeScript Open Source Need your help to make better edit code and commit changes

# Installations
```bash
$ git clone https://github.com/sanikava/Sanikava-Search
$ cd Sanikava-Search
$ yarn
$ yarn run start
```
it will be running at port 3000
http://localhost:3000
